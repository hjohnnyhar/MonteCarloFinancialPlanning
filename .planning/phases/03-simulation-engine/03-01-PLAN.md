    **GREEN phase:** Create `src/lib/simulation.ts` implementing the full engine per D-01 through D-05 and the research math:

    1. **Exports:**
       - `SimulationOverrides` interface: `{ annualSavingsRate?: number; retirementAge?: number; returnMean?: number; returnStdDev?: number; }`
       - `RETURN_ASSUMPTIONS` constant: `Record<RiskToleranceLevel, { mean: number; stdDev: number }>` with values: conservative `{ mean: 0.055, stdDev: 0.08 }`, moderate `{ mean: 0.07, stdDev: 0.12 }`, aggressive `{ mean: 0.09, stdDev: 0.16 }`
       - `INFLATION_RATE = 0.03`
       - `ITERATION_COUNT = 10_000`
       - `runSimulation(plan: FinancialPlan, overrides?: SimulationOverrides): SimulationResults`

    2. **randNormal(mean, stdDev):** Box-Muller with `Math.max(Number.EPSILON, Math.random())` guard for u1.

    3. **runSimulation logic:**
       a. Resolve assumptions: if overrides.returnMean/returnStdDev provided, use those; else use `RETURN_ASSUMPTIONS[plan.riskTolerance.level ?? 'moderate']`.
       b. Resolve annualSavingsRate: `overrides.annualSavingsRate ?? plan.income.annualSavingsRate`
       c. Compute `currentAge` from `plan.currentAge` (must be > 0, else throw Error "currentAge is required")
       d. Compute initialPortfolio: `plan.assets.checkingAndSavings + plan.assets.retirementAccounts + plan.assets.taxableInvestments`
       e. Compute annualSavings: `(plan.income.salary + plan.income.otherAnnualIncome) * resolvedSavingsRate`
       f. Sort goals chronologically: convert each goal to a `{ goal, targetYear }` tuple. For retirement: `targetYear = currentCalendarYear + (resolvedRetirementAge - currentAge)`. For purchase/education: use `goal.targetYear`. For legacy: `targetYear = Infinity` (processed last).
       g. Determine planHorizon: for retirement goals, `(resolvedRetirementAge - currentAge) + yearsInRetirement`. For non-retirement, `targetYear - currentCalendarYear`. Take the max across all goals. Default to 30 if no goals.
       h. For each of ITERATION_COUNT iterations:
          - Start with `portfolio = initialPortfolio`
          - Track `allGoalsMet = true` for this iteration
          - For each year from 0 to planHorizon-1:
            - Draw `r = randNormal(returnMean, returnStdDev)`
            - If year < yearsToRetirement (accumulation): `portfolio = portfolio * (1 + r) + annualSavings`
            - If year >= yearsToRetirement and retirement goal exists (decumulation): `portfolio = portfolio * (1 + r) - inflatedAnnualWithdrawal` where `inflatedAnnualWithdrawal = desiredAnnualIncome * (1.03)^yearsToRetirement`
            - Clamp portfolio to 0 if negative (per Pitfall 6)
            - At each goal's target year, check if portfolio >= inflated target amount. If not, mark that goal as failed for this iteration and set `allGoalsMet = false`.
          - Record per-goal success/failure counts and allGoalsMet count
       i. Compute results:
          - Per-goal: `probabilityScore = successCount / ITERATION_COUNT`, `fundingGap = median shortfall across failed iterations deflated to today's dollars by (1.03)^yearsToGoal` (if no failures, gap = 0), `targetAmount` = goal's original amount
          - Overall: `overallProbability = allMetCount / ITERATION_COUNT`
          - `ranAt = new Date().toISOString()`
          - `assumptions = { inflationRate: INFLATION_RATE, realReturnMean: returnMean, realReturnStdDev: returnStdDev }`

    4. Handle edge cases:
       - No goals: return overallProbability 1.0, empty goalResults array
       - Legacy goals with no targetYear: process at end of planHorizon, success = portfolio >= targetAmount at horizon end
       - Retirement age override (D-09): `const resolvedRetirementAge = overrides.retirementAge ?? (retirementGoal?.targetRetirementAge ?? 65)`

    Run tests — all 15 must pass.
