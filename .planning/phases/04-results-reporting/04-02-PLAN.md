# Plan 04-02: Enhanced simulation page UI with headline, recommendations, and projection

**Phase:** 04
**Sequence:** 02 of 03
**Requirements:** RSLT-01, RSLT-02, RSLT-03
**Depends on:** 04-01

## Goal

Upgrade the `/simulation` page with a color-coded headline score, a recommendations section showing 3 actionable items with specific amounts, and a collapsible year-by-year projection table.

## Context

Plan 04-01 extended `SimulationResults` with `scoreTier`, `recommendations[]`, and `yearlyProjection[]`. The existing `/simulation` page (`src/app/simulation/page.tsx`) already shows the headline percentage and goal breakdown. This plan enhances the headline visual (per D-03: color coding + tier label + explanation sentence), adds a recommendations card (per D-05, D-07: 3 computed recommendations with dollar amounts), and adds a collapsible year-by-year projection section (per D-11: hidden by default, expandable). The What-If panel remains unchanged (per D-02). All UI follows the established design system: raw Tailwind v4, Inter font, two weights (400/600), card pattern `rounded-lg border border-gray-200 bg-white p-6`, section headers `text-base font-semibold text-gray-900 mb-4`.

## Tasks

### Task 1: Polish headline score with color coding and tier label
- [ ] Update the "Overall probability of success" card in `page.tsx`: change the percentage text color based on `results.scoreTier.color` — green: `text-green-600`, amber: `text-amber-600`, red: `text-red-600`
- [ ] Add tier label below the percentage: `<p className="text-lg font-semibold {tierColor}">{results.scoreTier.label}</p>` (e.g., "Strong plan", "On track", "At risk")
- [ ] Add a one-liner explanation sentence below the label: "Your plan has an X% probability of meeting all financial goals across {N} simulated scenarios." using `text-sm text-gray-500`
- [ ] Keep the "Ran at" timestamp line at the bottom of the card

### Task 2: Add recommendations section and collapsible year-by-year projection
- [ ] Create `src/components/simulation/RecommendationsCard.tsx` — receives `recommendations: Recommendation[]` prop. Renders a card with header "Recommendations" and 3 recommendation items. Each item shows: a short summary sentence (e.g., "Increase monthly savings by $450 to reach 90%"), the lever name as a small label, and current vs suggested values using `formatCurrency`. Use numbered list (1, 2, 3) with `text-sm` body text. Card follows established pattern: `rounded-lg border border-gray-200 bg-white p-6`
- [ ] Create `src/components/simulation/YearByYearTable.tsx` — receives `projection: YearlySnapshot[]` and `planCurrentAge: number` props. Renders a collapsible section: header row "View year-by-year projection" with a chevron toggle (Unicode &#9660;/&#9650;), collapsed by default. When expanded, shows a table with columns: Year, Age, Portfolio Value, Savings, Withdrawals, Milestone. Goal milestone rows are highlighted with `bg-blue-50`. Use `formatCurrency` for money columns. Table uses `text-sm` with `border-collapse` and `border-b border-gray-100` between rows
- [ ] Import and render both new components in `page.tsx` inside the `{results && (<>...</>)}` block — place RecommendationsCard after the goal breakdown card, YearByYearTable after RecommendationsCard, and before the WhatIfPanel
- [ ] Extract `formatCurrency` and `formatGoalType` from `page.tsx` into `src/lib/formatters.ts` so they can be shared across page.tsx, RecommendationsCard, YearByYearTable, and later the PDF component. Update all imports
