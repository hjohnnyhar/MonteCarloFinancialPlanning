```typescript
interface GoalFormProps {
  initialData?: GoalType;  // for editing; undefined for new goal
  onSave: (goal: Goal) => Promise<void>;
  onCancel: () => void;
}
```

**Common pattern per form:**
1. Use `useForm` with `zodResolver` and the matching goal schema
2. If `initialData` provided, use as defaultValues (editing mode)
3. If no initialData, use sensible defaults for new goal
4. Form renders as `<form onSubmit={handleSubmit(onFormSave)}>` where onFormSave calls `onSave(data)`
5. Two buttons at bottom: "Discard Changes" (calls onCancel) and "Save Goal" (submits form)
6. All currency/numeric fields use `type="text" inputMode="decimal"` per UI-SPEC
